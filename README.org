Hello! This is my [[https://nixos.org/][NixOS]] configuration that I use to maintain my machines. 

The machines:
- ~lenovo~
- ~server~

I use [[https://orgmode.org/][org-mode]] in [[https://www.gnu.org/software//emacs/][emacs]] to manage my NixOS configuration.

To use this repo:
- First clone the repo
- Assuming you've emacs installed in your nixos system, run the command below
#+begin_src bash :tangle no
   emacs README.org --batch -f org-babel-tangle
#+end_src
   The above will generate all the .nix files.
- Then run the nixos-rebuild command
#+begin_src bash :tangle no
   sudo nixos-rebuild switch --flake .#<lenovo/server>
#+end_src
- For [[https://github.com/nix-community/home-manager][Home-Manager]] configuration, run the following while being in the repo directory.
#+begin_src bash :tangle no
   home-manager switch --flake .#kin-lenovo
#+end_src
   I only have one home-manager user configured, and that is for my ~lenovo~ machine.
   
* The Flake File
In NixOS flakes are used to package and manage Nix projects, flakes are used as a way to produce reproducible builds as well as controlling multiple systems.

I also use flakes as a way to import my non-flake files and use it to import files within the various non-flake git repositories, example of this last statment is the input named ~walls~ that fetches from [[https://codeberg.org/kinest/walls][my repo]] that stores wallpapers, this is later used in [[modules/features/wm/niri.nix#L218][Niri]] along with [[https://swaybg.com/][Swaybg]].

*** Whatmp3
I have packaged whatmp3 as a nix package.
This is used to transcode flac files to mp3 [320, V0], also used to downsample flac from 24 bit to 16 bit.

To use this program, after installing run this:
#+begin_src bash
  whatmp3 -rwdz -o <outputdir> -s "RED" -t <tracker announce url> --FLAC --320 --V0 <flac dir>
#+end_src

| Flag   | Action               |
|--------+----------------------|
| -r     | replaygain           |
| -w     | overwrite            |
| -z     | zeropad tracknumbers |
| -d     | dither to 16/44.1    |
| -s     | Source              |
| --320  | Convert to mp3 320   |
| --V0   | Convert to mp3 V0    |
| --FLAC | Convert to flac      |
